checkbox

Un checkbox en un design system es un elemento de interfaz que permite seleccionar opciones de una lista. Se define como un componente estándar en el sistema de diseño para mantener la coherencia y mejorar la experiencia del usuario.

Datalist

Campo interactivo que permite la selección de una opción además de poder escribir con texto predictivo.
