Links

Un link (enlace) en un design system es un elemento estándar para la navegación del usuario hacia otros recursos o páginas.

Links

Los links son elementos interactivos de la interfaz que permiten al usuario identificar una acción que lo llevará a otra página o abrirá otra capa de navegación para continuar o ampliar su experiencia en la web.

link-line

Link básico

link-line 

	<a class="link-line" href="">
		link-line
	</a>

link-raw

Link con una línea o múltiple

link-raw 

	<a class="link-raw" href="">
		link-raw
	</a>

link-svg-pre

Link con icono svg-icon al inicio

 link-svg-pre 

	<a class="link-line link-svg-pre" href="">
		link-svg-pre
		<svg-icon src="alert"></svg-icon>
	</a>

link-svg-post

Link con icono svg-icon al final

link-svg-post 

	<a class="link-line link-svg-post" href="">
		link-svg-post
		<svg-icon src="alert"></svg-icon>
	</a>
