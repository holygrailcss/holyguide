Types en inputs

El atributo type en un elemento input especifica el tipo de entrada que se espera del usuario. Esto determina el tipo de teclado virtual o de entrada que se mostrará en dispositivos móviles y puede cambiar la forma en que se procesan los datos introducidos por el usuario. Algunos de los tipos más comunes de input son: 

  
<input type="text"> (default value) 
 //para entrada de texto
<input type="number"> 
 // entrada numérica
<input type="email"> 
 //para entrada de direcciones de correo electrónico @
<input type="password"> 
  //para entrada de contraseña
<input type="date"> 
 // entrada de fechas
<input type="radio"> 
 //para seleccionar una opción única de un conjunto de opciones
<input type="checkbox"> //para seleccionar varias opciones de un conjunto de opciones
<input type="button">  

<input type="color">

<input type="submit"> 

<input type="tel"> 

<input type="time"> 

<input type="url">
