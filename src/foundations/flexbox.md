| .mdmax:flex | Flex hasta md -1 |
| .lgmax:flex | Flex hasta lg -1 |
| .sm:flex    | Flex desde sm    |
| .md:flex    | Flex desde md    |
| .lg:flex    | Flex desde lg    |
