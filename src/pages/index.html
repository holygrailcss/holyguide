Introducción

Aqui veremos como customizar el holygrail de cero siguiendo la especificación de Holygrail css, desarrollamos una biblioteca de elementos de usuario que contiene un conjunto de componentes y demostraciones de alta calidad para crear interfaces de usuario Previas a Desarrollo.


Instalación

Aqui veremos como customizar el holygrail de cero siguiendo la especificación de Holygrail css, desarrollamos una biblioteca de elementos de usuario que contiene un conjunto de componentes y demostraciones de alta calidad para crear interfaces de usuario Previas a Desarrollo.

npm i holygrailcss

Descargar el proyecto


Opción 1: Instalación con NPM package


Recomendamos el uso de npm , a fin de mantener la última versión estable de holygrail

npm i holygrailcss

Opción2: Clonar el repositorio de Github


Holygrail está disponible con github

git clone https://github.com/holygrailcss

Opción3: Link directo desde cdn


Holygrail está disponible en jsdelivr

https://cdn.jsdelivr.net/npm/holygrailcss@1.8.8/dist/style.css

Ejecutar el proyecto

Antes que nada comprueba que tienes node.js en tu ordenador, en caso de no tenerlo instalarlo.

0-Comprobar versión de node

node -v

1- Una vez en la carpeta del proyecto instalamos los módulos necesarios con npm

npm i

2- Compilamos sass con npm

npm build-sass

3- Usamos Gulp para comprimir y lanzar el proyecto

gulp style gulp watch

Si no tienes gulp instalado:

npm install --save-dev gulp

Si tienes problemas en Mac puede que necesites ejecutar el comando como administrador

sudo npm install --save-dev gulp
