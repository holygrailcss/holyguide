IMPORTAR

Agregar ~holygrailcss/scss/style.scss desde node_modules a tu archivo de entrada de scss: 

@import "setup.scss"; 
@import '~holygrailcss/scss/style.scss';

COMPATIBILIDAD CON NAVEGADOR

El autor ha intentado agregar prefijos de proveedores apropiados a algunos estilos, solo tienes que asegurarse de que las propiedades CSS que estás utilizando actualmente sean compatibles con la mayoría de los navegadores modernos - caniuse.com

INSTALACIÓN MANUAL

0. Comprobar node

Antes que nada comprueba que tienes node.js en tu ordenador, en caso de no tenerlo instalarlo.

node -v

Una vez en la carpeta del proyecto instalamos los módulos necesarios con npm

npm i

Compilamos sass con npm

npm build-sass

Usamos Gulp para comprimir y lanzar el proyecto

gulp style gulp watch

Si no tienes gulp instalado:

npm install --save-dev gulp

Si tienes problemas en Mac puede que necesites ejecutar el comando como administrador

sudo npm install --save-dev gulp

1. Clonar el repo

Holygrail está disponible con github

git clone https://github.com/holygrailcss

2. Ejecutar proyecto

gulp
